# nobug-server
>A website to manage bugs.

## Technology Stack
Node.js + Koa2 + Mysql

## Start
```bash
# You have to install redis
brew install redis

# start redis server
redis-server

# if you wanna to get redis detail
redis-cli

# start mysql
sudo /usr/local/MySQL/support-files/mysql.server start

# install
cnpm install

# start node server
npm run server

```
